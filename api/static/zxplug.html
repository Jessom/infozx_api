# zxplug

使用插件，需在对应的js文件中加入以下代码

import zxplug from '@/common/plug/zxplug'
Vue.use(zxplug)

# loading

this.$loading({msg:'加载中...'})        // 打开loading
this.$loading.close()  // 关闭loading
参数类型描述默认值
msgStringloading提示文字''
maskBoolean是否显示遮罩true
typeStringloading样式，可取值scale, circle'circle'

# nothing

this.$nothing()        // 显示nothing
this.$nothing.close()  // 关闭nothing
参数类型描述默认值
msgString提示文字''
