# 快速开始

提醒

所有组件均以z-开头, 除非重写组件样式, 其他情况请不要使用z-开头的class

警告

DEMO是用单页面写的，样式方面会有部分问题，请勿介意

# 说明

  • 基于vue cli搭建的多页面开发环境
  • scss、 sass预处理css文件
  • Vue版本 2.4.2

# Build Setup

使用vuepress编写项目文档

# 安装依赖
npm install

# 开发环境
npm run dev

# 编译
npm run build

# 编写文档
npm run docs:dev

# 编译文档
npm run docs:build

# 编译项目 && 编译文档 && 提交码云
npm run deploy
# or
bash deploy.sh
