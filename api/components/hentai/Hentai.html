# refs

事件返回描述
initarray初始化数据，查看array

# array

提示

array中的项，可以随意添加

参数名类型描述
idNumber必须参数
titleString渲染的标题
childrenArray渲染的子部门，同array
staffArray渲染的成员列表，staff

# staff

提示

staff中的项，可以随意添加

参数名类型描述
idNumber必须参数
titleString渲染的标题
imageString成员头像
descString成员描述

# events

事件参数说明
backself点击返回的回调
closeself点击关闭的回调
success{ select, [actived], [include] }点击确定的回调，查看详情

# success

  • select: 1全部、2本人、3部门、4成员
  • actived: 当select==3 || select==4携带该参数，表示选择的部门或成员
  • include: 当select==3携带该参数，表示是否包含子部门
    • true: 包含子部门
    • false: 不包含子部门
