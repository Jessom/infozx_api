# DoubleHit

Demo

严重警告

这是一个比较ド へんたい的东西；因为内部使用了较多的递归，所以你应该懂了

# 功能介绍

组件有三行

  • 第一行选择部门
  • 第二行根据所选部门，从没选的部门中选择包含哪些成员
  • 第三行根据所选部门，从选择的部门中选择排除哪些成员

# props

参数名类型默认值描述
listArray-渲染数据, 查看list详情
showIncludeBooleantrue选择时，是否可以包含子部门
isBranchBooleantrue是否选中部门
isIncludeBooleantrue是否选中包含谁
isExcludeBooleantrue是否选中排除谁

# list

必须参数

注意： 请不要出现include、disabled、flag字段，这俩字段，在操作数据时会发生改变

参数名类型默认值描述
titleString-部门名
idNumber-就是id
staffArray-员工
childrenArray-子部门

# staff

参数名类型默认值描述
titleString-员工名
idNumber-就是id
descString-员工描述信息
imageString-头像url

# children

同list

# refs

事件参数说明
getData{branch, exclude, include}获取选择信息
clear-清除选中信息
  • belong: [{children, id, staff, title, flag}]
    • flag: true为包含子部门，false为不包含子部门

# events

事件参数说明
complete{branch, exclude, include}点击确定按钮后的回调
