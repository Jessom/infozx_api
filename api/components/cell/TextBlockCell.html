# 示例










 



































<template>
	<div style="padding:15px 0;background:red">
		<text-block-cell
			:list='list'
			@onCellClick='onCellClick' />
	</div>
</template>

<script>
import TextBlockCell from '@/components/Cell/TextBlockCell.vue'
export default {
	data() {
		return {
			list: [{
				desc: '按时出勤',
				title: '0天'
			}, {
				desc: '缺勤',
				title: '7天'
			}, {
				desc: '异常出勤',
				title: '3天'
			}, {
				desc: '请假出差',
				title: '0天'
			}, {
				desc: '休息',
				title: '2天'
			}]
		}
	},
	methods: {
		onCellClick(obj) {
			console.log(obj)
		}
	},
	components: {
		TextBlockCell
	}
}
</script>

# props

参数名类型默认值描述
listArray-渲染数据, 查看list详情
optionsObject-配置参数, 查看options详情

# list

参数名类型默认值描述
titleString-上面的文字
descString-下面的文字

# options

参数名类型默认值描述
borderBooleantrue是否显示边框
themeBooleantrue是否主题背景色，ture 主题背景白字，false 透明背景黑字

# events

事件参数说明
onCellClick{title,desc}点击事件
