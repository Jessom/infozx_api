# 神奇的calc()

亲测，支持在IOS8.1系统(因为这里设备最低是IOS8.1)

# 语法

calc() = calc(四则运算)

# 说明

用于动态计算长度值
需要注意的是，运算符前后都需要保留一个空格，例如

width: calc(100% - 10px)

任何长度值都可以使用calc()函数进行计算
calc()函数支持+ - * / 运算
calc()函数使用标准的数学运算优先级规则

# 临床试验

在写addImgButton组件时
我是希望一排平均显示4个图片容器，每个容器的宽高相同
首先想到的就是像素，但是像素存在一个明显的缺点，不同设备分辨率不同，导致无法平均分布，抛弃
第二个想到的就是百分比，缺点也很明显，高度不好设置，抛弃

一道闪电， 真実はいつも一つ
calc这个东西被这道闪电劈了下来
劈下来的不止有calc，还有vm、vh这俩货
事不宜迟，赶快开动(万一忘了呢) 

.z-img-item {
	width: calc(25vw - 17.5px);
	height: calc(25vw - 17.5px);
	line-height: calc(25vw - 17.5px);
}
